Connects to Redis from Ballerina

### Package Overview
#### Compatibility
|                                                                                    | Version               |
|------------------------------------------------------------------------------------|-----------------------|
| Ballerina Language Version                                                         | **Swan Lake Beta 2**  |

The `redis` is a [Ballerina](https://ballerina.io/) connector for Redis.

This package provides the capability to easily access Redis.
